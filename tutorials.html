Tutorial 1

Robotics in 2025 – Where I start

In this tutorial, I outline how I see robotics in 2025 – available hardware, cheap sensors, open-source libraries, and what is realistic to build as a student or solo developer.

  1. Decide one small project (line follower, arm, or simple rover).
  2. Pick components that are easy to buy around you.
  3. Choose one language/platform (Arduino, ROS, Python, etc.).
  4. Break the project into tiny tasks and write daily notes here.

Tutorial 2

Working on a project – simple flow

This tutorial is just a simple workflow I can follow whenever I start any robotics or automation project.

  1. Write down the goal in one clear sentence.
  2. Draw a very rough block diagram (inputs → processing → outputs).
  3. Setup basic hardware and test each part individually.
  4. Combine hardware with simple code first, then improve step by step.
  5. Record short videos and paste them into blogs or tutorials.
